# Project Outlines

- Maximise coding, minimise manual work (design, map making etc)
- Game development in ~5 weeks
- Plan expansions for extra time

## Game Ideas

### Idea 1 - Contra

Inspiration:
- Contra
- Arcade shooters

Fundamentals:
- Side-scrolling action
- Run-and-gun shooting
- Enemy waves and boss fights

Nice to haves:
- Co-op mode
- Weapon upgrades

### Idea 2 - Gold Miner

Inspiration:
- Gold Miner
- Casual arcade games

Fundamentals:
- Hook-and-reel mechanic
- Time-based levels
- Score and item system

Nice to haves:
- Power-ups
- Endless mode
